# Screen Specification

## Authentication Screens

### Splash Screen
- Display app logo and branding
- Auto-navigate to Login or Main App based on auth state

### Login Screen
- Email input field
- Password input field (with show/hide toggle)
- "Forgot Password?" link
- Login button
- Navigate to Register option

### Register Screen
- Full name input field
- Email input field
- Password input field (with show/hide toggle)
- Confirm password input field
- Register button
- Navigate to Login option

### Forgot Password Screen
- Email input field
- Send OTP button

### OTP Verification Screen
- 6-digit OTP input
- Resend OTP option
- Verify button

### Verify Email Screen
- Instruction message
- Resend verification email button
- Confirm button

---

## Main Application Screens

### Newsfeed Screen
- Stories row (horizontal scroll)
- Post feed (vertical scroll)
  - Post card: author info, image/video, caption, like/comment/share/save actions
- Floating action button for Create Post

### Post Detail Screen
- Full post view
- Comments section
- Like, Comment, Share, Save actions

---

### Library Screen
- Search bar
- Category filter chips (horizontal scroll)
- Tutorial grid/list
- Difficulty level badge on each tutorial card

### Tutorial Detail Screen
- Cover image/video
- Title, author, difficulty, duration
- Description
- Steps overview
- Start Learning button
- Save/Bookmark button

### Step-by-Step Screen
- Step counter (e.g. Step 1 of 10)
- Step image/video
- Step description
- Previous / Next navigation buttons
- Progress bar

### Achievement Screen
- Total XP points
- Achievement badges grid
- Progress bars for ongoing achievements

---

### Creator Studio Screen
- Tabs: My Posts | My Tutorials | Submissions
- Create Post button
- Create Tutorial button

### Create Post Screen
- Image/video picker
- Caption input
- Tags input
- Publish button

### Create Tutorial Screen
- Title input
- Difficulty selector
- Estimated duration input
- Cover image picker
- Steps editor (add/edit/delete steps)
  - Each step: image + description
- Submit for Review button

### Submission Detail Screen
- Tutorial preview
- Submission status (Pending / Approved / Rejected)
- Reviewer feedback (if any)

---

### Messages Screen
- Conversation list
- Search conversations

### Profile Screen
- Avatar, name, bio
- Stats: Tutorials created, Posts, Followers, Following
- Tabs: Saved Tutorials | Saved Posts | Achievements
- Edit Profile button
- Settings button

### Settings Screen
- Account settings
- Notification preferences
- Privacy settings
- Logout button
