# Origami Learning Platform

## Overview

Origami is a mobile application that helps users:

* Learn origami through step-by-step tutorials
* Share creations with the community
* Publish tutorials
* Interact through comments and reactions
* Build achievements and experience points

## Main Modules

### Authentication

* Login
* Register
* Forgot Password
* Email Verification

### Community

* Newsfeed
* Like Post
* Comment Post
* Share Post
* Save Post

### Learning

* Library
* Tutorial Details
* Step-by-Step Learning
* Achievement System

### Creator Studio

* Create Tutorial
* Create Community Post
* Manage Submitted Content

### User Profile

* Personal Information
* Saved Tutorials
* Saved Posts
* Achievements
* Settings
